## 🧠 ¿Qué es un Dashboard de Business Intelligence?

Un dashboard de Business Intelligence (BI) es una interfaz visual diseñada para consolidar y representar datos de múltiples fuentes en un solo entorno interactivo. Su propósito es transformar información compleja en análisis claros y accionables que faciliten la comprensión del comportamiento del sistema y apoyen la toma de decisiones estratégicas.

---

## 🎯 Objetivo

El objetivo del dashboard es centralizar la información del sistema de farmacia en un entorno visual y analítico, permitiendo observar de manera clara el comportamiento operativo del sistema y facilitando la toma de decisiones basada en datos reales y actualizados.

---

## 🔗 Integración de Datos

El dashboard se construye sobre una arquitectura híbrida de datos que combina **MySQL** y **MongoDB** para ofrecer una visión más completa del sistema.

- 🗄️ **MySQL**: almacena la información estructurada del sistema, como inventarios, lotes, recetas, usuarios y movimientos operativos.
- 📄 **MongoDB**: gestiona datos no estructurados como  eventos del sistema y registros dinámicos.

Esta integración permite generar visualizaciones más ricas dentro de Navicat BI, combinando datos operativos y analíticos para un análisis más profundo del sistema de farmacia.
