## Qué es un Schema de Base de Datos

Un schema de base de datos es la estructura completa y organización lógica de una base de datos. Define cómo están organizadas las tablas, sus campos, tipos de datos, restricciones, relaciones entre tablas e índices. Es el blueprint o plano arquitectónico que determina cómo se almacenan y relacionan los datos en el sistema.

El propósito principal es establecer una estructura coherente, eficiente y escalable que garantice integridad de datos, optimice consultas y facilite el mantenimiento del sistema de información.
