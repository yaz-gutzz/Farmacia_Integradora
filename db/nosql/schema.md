## Qué es un Schema en MongoDB

Un schema en MongoDB es la estructura lógica y organización de las colecciones y documentos dentro de la base de datos NoSQL. Define cómo están organizadas las colecciones, sus campos, tipos de datos, validaciones y relaciones entre documentos. Es el blueprint o plano arquitectónico que determina cómo se almacenan y relacionan los datos en el sistema.

El propósito principal es establecer una estructura coherente, eficiente y escalable que garantice integridad de datos, optimice consultas y facilite el mantenimiento del sistema de información.
