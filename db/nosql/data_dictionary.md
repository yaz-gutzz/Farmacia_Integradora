## Qué es un Diccionario de Datos

Un diccionario de datos es una herramienta de documentación que describe la estructura, contenido y características de una base de datos NoSQL. Proporciona una referencia completa sobre todas las colecciones, campos, tipos de datos y relaciones que componen el sistema de información.

El propósito principal es facilitar la comprensión, mantenimiento y uso consistente de los documentos dentro de la organización, asegurando que todos los usuarios entiendan qué información contiene la base de datos, cómo se relacionan las entidades y cuáles son los estándares de calidad y seguridad aplicados.
