# Sistema de Gestión Farmacéutica Hospitalaria

Plataforma integral para gestión de medicamentos, inventarios, compras y dispensaciones en hospitales.

---

## 📁 Estructura

| Carpeta | Descripción |
|---------|-------------|
| **DOCS/** | Documentación: FRs, NFRs, BRs |
| **DB/SQL/** | Scripts y documentación MySQL |
| **DB/NOSQL/** | Scripts y documentación MongoDB |
| **DB/SECURITY/** | Políticas de seguridad |
| **API/** | Backend (FastAPI) |
| **TESTS/** | Suite de 10 pruebas |
| **DASHBOARD/** | Dashboards Navicat BI |

## 🛠️ Tecnologías

- **Backend:** Python, FastAPI, SQLAlchemy
- **BD:** MySQL, MongoDB
- **API:** REST, JWT, Swagger
- **Testing:** Pytest
- **BD Adicional:** Navicat BI
